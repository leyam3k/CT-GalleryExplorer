# CT-GalleryExplorer

An advanced gallery extension for SillyTavern/CozyTavern that allows you to browse, manage, and view character images seamlessly.

## Features

- **Media Grid**: View all images and videos associated with a character or folder.
- **Lightbox Viewer**: Full-screen media viewer with zoom, pan, and navigation support.
- **Media Management**:
  - Delete unwanted media.
  - Upload new images/videos directly to the character's folder.
  - Copy media as Markdown or HTML.
- **Sorting**: Sort by Newest, Oldest, or Name (A-Z, Z-A).
- **Integration**:
  - Adds a button to the Sidebar (via CT-SidebarButtons).
  - Adds a button to the Character Card menu for quick access.
  - Syncs with the currently selected character.

## Installation

1. Open SillyTavern.
2. Go to `Extensions` > `Install Extension`.
3. Paste the repository URL or install from the list if available.
4. Reload SillyTavern.

## Usage

- **Opening the Gallery**:
  - Click the **Image Gallery** button in the Extensions menu.
  - Use the Sidebar button (if configured).
  - Click the Gallery icon in the Character Card menu (top right of the character panel).
- **Navigation**:
  - Click a folder to browse its contents.
  - Click an image to open the Lightbox viewer.
  - Use arrow keys (Left/Right) to navigate images in Lightbox.
  - Use Escape to close the Lightbox or the Gallery.
- **Selection Mode**:
  - Enable selection mode to select multiple images.
  - Bulk delete or download selected images.

## Requirements

- SillyTavern (latest version recommended)
- Optional: [CT-SidebarButtons](https://github.com/CozyTavern/CT-SidebarButtons) for sidebar integration.
